## ğŸš€ Getting Started

### Prerequisites

- **Node.js**: v18.0.0 or higher
- **npm**: v9.0.0 or higher
- **Expo Go**: Available on iOS App Store and Google Play Store for physical device testing.

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Atifhasan250/monthly-todo-planner.git
   cd monthly-todo-planner
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Launch the Development Server**
   ```bash
   npm run expo:dev
   ```

### Running the App

- **Mobile (Recommended)**: Scan the QR code displayed in your terminal using the Expo Go app.
- **Web**: Press `w` in the terminal to open the application in your default browser.
- **Android/iOS Simulators**: Press `a` or `i` respectively (requires local setup).
